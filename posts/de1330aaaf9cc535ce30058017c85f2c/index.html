VS2022 .netcore启用本地代码调试

调出项目属性界面

在项目上点击右键，选择Properties(属性)页面。

选择Debug属性

选中Debug分项中的General后点击Open debug launch profiles UI链接。

启用本地代码调试

选择Enable native code debugging启用本地代码调试。
