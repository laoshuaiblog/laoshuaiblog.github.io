介绍

Dev C++ 是一款轻量化的 C / C++ 桌面 IDE ，由于不需要额外配置 GCC 、即开即用的特点，因此常常用于 C / C++ 语言基础的学习。

目前官方最新版为 Dev-C++ 5.11。
