Google Cloud 免费层级

Google Cloud 免费层级提供了各种免费资源，可让您通过自行试用了解 Google Cloud 服务。无论是想学习该平台基础知识的全新客户，还是想体验新解决方案的老客户，Google Cloud 免费层级都能满足您的需求。

Google Cloud 免费层级包含两个部分：

  • 12 个月免费试用，含 $300 赠金，可用于试用任意 Google Cloud 服务。
  • “始终免费”计划，可让您限量免费使用很多常用 Google Cloud 资源。

本文介绍了 Google Cloud 免费层级的这两个部分。

注意：如果您对 Google Cloud 免费层级还有其他疑问，请联系支持团队。

获享 $300 赠金，免费试用 12 个月

免费试用提供了一定的赠金，在您学习 Google Cloud 知识期间可用于支付相关资源使用费。

参与计划的资格要求

只要符合以下条件，即有资格参与本计划：

  • 您从不是 Google Cloud 付费客户。
  • 您以前没有注册过免费试用。
  • 您已经通过 Google Cloud Console 为自己的帐号启用了结算功能。
  • 您并未与销售代表合作设立附带条款清单的企业帐号。

如果您需要设立企业帐号，请联系销售团队。

计划的启动

设置好第一个结算帐号后，包含 $300 赠金的 12 个月免费试用即自动开始。

您必须提供信用卡或银行详细信息才能设置结算帐号并验证您的身份，但在免费试用期间我们不会向您收取费用。

计划的覆盖范围

您的免费试用赠金适用于所有 Google Cloud 资源，但有以下例外情况：

  • 您不能同时运行 8 个以上的内核（或虚拟 CPU）。
  • 您不能向虚拟机实例添加 GPU。
  • 您不能申请增加配额。如需大致了解 Compute Engine 配额，请参阅资源配额。
  • 您不能创建基于 Windows Server 映像的虚拟机实例。

您必须升级帐号才能执行上述操作。

免费试用赠金还可用于 Google Maps Platform。

受限制的操作

除资源限制外，免费试用服务条款还指出了在免费试用期间禁止的使用场景。 例如，您不得在免费试用期间使用 Google Cloud 服务进行加密货币挖矿活动。

计划的时长

只要符合以下条件之一，您的免费试用即告结束：

  • 注册时间满 12 个月。
  • 消费完 $300 赠金。

在免费试用期内，您的剩余资金和剩余试用天数均显示在 Google Cloud Console 顶端。

服务条款

您必须接受免费试用服务条款和 Google Cloud 服务条款。

服务等级协议

免费试用期间不适用服务等级协议。 因为免费试用期是您测试和探索 Google Cloud 的时期，此期间不应在 Google Cloud 上运行生产应用。

使用“始终免费”的资源

在免费试用期间使用“始终免费”范围内的资源时，不会消耗您的免费试用赠金。

信用卡验证

当您注册免费试用时，Google 会要求您提供信用卡信息。Google 会将您的信用卡信息用于以下用途：

  • 验证您的身份。
  • 区分真实客户与机器人。

即使您设置了结算帐号，只要您不升级到付费帐号，我们就不会向您收费。注册后，您可能会在银行对帐单上看到一笔由 Google 汇入的 0.00 到 1.00 美元之间的存款交易。这笔交易具有以下属性：

  • 这笔交易属于一条旨在验证您的帐号的授权请求，而不是永久性费用。

  • 您的银行可能会将这笔交易的款项转为您当地的货币。

  • 这笔交易可能会在您的对帐单上显示长达一个月的时间，之后这笔款项才会被撤回。

如果您对授权有任何疑问，请与您的银行联系。

注意：如果您在验证过程中遇到问题，请与结算支持团队联系。

没有信用卡？

如果您没有信用卡，可以提供银行账户信息作为替代，如果您选择提供银行账户信息：

  1. Google 会随机向您的银行帐户汇入一笔 1 美元以内的款项，这一步需要 2 到 3 天时间。
  2. 您使用这笔汇入的款项完成注册过程。

免费试用结束

免费试用将在您用完所有赠金或注册满 12 个月后（以先发生者为准）结束。在试用期结束之时，以下条件适用：

  • 您必须升级为付费帐号才能继续使用 Google Cloud。
  • 您在试用期间创建的所有资源都将无法使用。

  • 存储在 Compute Engine 中的所有数据都会丢失。

  • 您的帐号将进入为期 30 天的宽限期，在此期间您可以恢复试用期存储在任何 Google Cloud 服务中的资源。

  • 您可能会收到一条消息，告知您的帐号已被取消，但这仅表示您的帐号已被暂停使用，以免产生费用。

恢复数据

如需导出您在试用期间存储在 Google Cloud 服务（Compute Engine 除外）中的所有数据，请与 Google Cloud Platform 结算支持团队联系。您的数据和资源仅在免费试用结束后的 30 天内可用。

注意：您在 Compute Engine 中创建的虚拟机实例上所用的数据无法自动恢复。您必须在试用期结束之前，从 Compute Engine 虚拟机实例中手动导出希望保留的任何数据。

升级为付费帐号

在试用期结束后，您必须升级为付费帐号才能使用 Google Cloud。要利用付费帐号的功能（比如使用 GPU），您可以在试用期结束之前升级。在升级时，以下条件适用：

  • 您的帐号中会保留所有剩余的、未过期的免费试用赠金。
  • 超过剩余赠金额度的资源使用费将由您登记的信用卡支付。

您可以在开始免费试用后随时升级您的帐号。根据您升级的时机不同，以下条件适用：

  • 如果您在试用期结束之前升级，则剩余赠金将计入您的付费帐号。在这种情况下，您可以不间断地继续使用在免费试用期内创建的资源。

  • 如果您在试用期结束后的 30 天内升级，那么可以恢复试用期间创建的资源。

  • 如果您在试用期结束超过 30 天后才升级，您在免费试用期间的资源将全部丢失。

升级帐号

您可以通过 Google Cloud Console 从免费试用升级到付费帐号。要将您的帐号升级为付费帐号，请点击激活按钮。如果看不到激活按钮，请点击菜单栏上的免费试用状态图标 ，激活按钮即会出现。

您必须是帐号的结算管理员才能进行此项更改。

免费试用结束后的费用

Google Cloud 服务仅对您使用的资源收费。每项服务都有自己的价格模型，详情请参见各项服务的对应文档。

您可以估算 Google Cloud 的使用费，可以使用价格计算器将预计要使用的资源换算为每月的预估使用费，也可以查询价格页面。
