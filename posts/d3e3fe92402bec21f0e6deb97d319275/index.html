Android Studio教程从JDK环境配置到AS安装及常用设置（图文详解）

安装 JDK

JDK官网下载需要账号，可直接点击下载 ： java8u201
新版Android Studio可以不安装JDK，可省略

下载完双击安装


 默认会下载JRE(Java运行环境)

配置JDK环境变量

  1. 右键此电脑/计算机，点击属性，点击高级设置

  2. 点击环境变量，在系统变量中新建 JAVA_HOME

  3. 在系统变量中找到path变量，点击编辑-新建：%JAVA_HOME%\bin ，%JAVA_HOME%\jre\bin （直接copy）。添加到path变量中，之后点击确认。

  4. 测试是否配置成功：打开cmd（命令提示符）,分别输入Java，java

安装Android Studio

下载地址

  1. 点击下载最新版本，选择其他版本（DOWNLOAD OPTIONS），我直接下的最新的3.4
  2. 下载完成，点击安装

  1. 点击Next，下个界面继续Next，自定义安装路径或者默认
  2. 创建快捷方式（默认就行），继续等待安装

  1. 安装完成，勾选上start Android Studio 点击Finish

  1. 分别是设置为之前版本 ，自定义设置，不进行设置，勾选Do not import settings 点击Ok

  1. 点击 send usage statistics to Google （允许google收集数据）

  2. 没有Android SDK，点击Cancel

  3. 设置类型，默认就行，点击Next

  4. 选择主题 黑色和白色，选择后点击Next

  5. 下载android studio 所需要的资源 点击 Finish，等待下载完成

  6. Android Studio 安装完成

AS常用设置

  1. 设置字体大小：点击右上角 File–>Settings–>Editor–>Font设置size–>apply–>ok
  2. AS修改快捷键方式：点击右上角 File–>Settings–>KeyMap设置–>apply–>ok
  3. 设置代码提示忽略大小写：点击右上角 File–>Settings–>Editor–>General–>Code Completion设置–>apply–>ok
  4. 设置自动导包和删包：点击右上角 File–>Settings–>Editor–>General–>Auto Import把下边图片上的勾选上–>apply–>ok
  5. 设置成员变量首字母加m，静态变量加首字母s：点击右上角 File–>Settings–>Editor–>Code Style–>java按图片设置–>apply–>ok
