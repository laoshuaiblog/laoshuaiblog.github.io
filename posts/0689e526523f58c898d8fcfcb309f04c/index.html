快速画流程图

使用在线工具：PlantUML

PlantUML
官网:https://plantuml.com/
中文官网:https://plantuml.com/zh/

使用步骤如下：

1、拷贝一个完成函数：

int func_init(const char *tag) {
    if (tag != NULL) {
        printf("set TAG :%s", tag);
    }

    print_test();
    return 0;
}

2、使用chatGPT把代码转换成标准的plantuml语言输出流程图，具体输入如下：

用标准plantuml语言输出流程图
int func_init(const char *tag) {
if (tag != NULL) {
printf(“set TAG :%s”, tag);
}
print_test();
return 0;
}

3、chatGPT输出如下：

@startuml
start
if (tag != NULL) then (yes)
:printf(“set TAG :%s”, tag);
else (no)
endif
:print_test();
stop
@enduml
