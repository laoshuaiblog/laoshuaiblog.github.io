kali安装微信

下载地址：https://github.com/geeeeeeeeek/electronic-wechat/releases
tar -zxvf linux-x64.tar.gz -C /opt
vim /usr/share/applications/webchat.desktop

[Desktop Entry]
Encoding=UTF-8
Version=v2.0
Name=wechat
GenericName=wechat
Comment=wechat_web
Exec=/opt/electronic-wechat-linux-x64/electronic-wechat
Icon=/opt/electronic-wechat-linux-x64/wechat.png
Terminal=false
Type=Application
Categories=Application;Utility;Network;InstantMessaging;
StartupNotify=false
  • 常见错误及解决：
R: ./electronic-wechat: error while loading shared libraries: libgconf-2.so.4: cannot open shared object file: No such file or directory
P: sudo apt install libgconf-2-4
