五、Nginx代理Jenkins

1、查看jenkins配置文件位置

systemctl status jenkins

2、修改配置文件中的访问前缀 

vim /usr/lib/systemd/system/jenkins.service

        找到JENKINS_PREFIX，然后取消注释，将它修改为我们想要使用的前缀即可，这里我们使用默认的jenkins演示，修改完成后，保存退出，重启Jenkins。

3、修改nginx配置文件

        在nginx服务器中增加Jenkins的访问代理即可。

location /jenkins {
      proxy_pass http://ip:port;
}
