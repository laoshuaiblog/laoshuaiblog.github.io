MultipartFile 和 file 的区别在于：

  1. 定义不同：MultipartFile 是Spring 框架中的一个类，而 file 是 Java 标准库中的一个类。
  2. 用途不同：MultipartFile 主要用于上传文件，而 file 主要用于读写文件。
  3. 类型不同：MultipartFile 是一个接口类型，而 file 是一个基本类型。
  4. 读写方式不同：MultipartFile 可以读取二进制数据和文件名称，而 file 只能读取文件名称。
  5. 处理方式不同：MultipartFile 可以使用 Spring 框架提供的方法进行处理，而 file 需要使用 Java 标准库提供的方法进行处理。
