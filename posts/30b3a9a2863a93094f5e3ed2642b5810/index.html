面试题（算法题） - 百钱买百鸡

  • 题目
  • 思路
  • 代码
  • 答案

题目

条件：现有 100 钱，一共要买公鸡、母鸡、小鸡三种鸡，已知公鸡 5 钱一只，母鸡 3 钱一只，1 钱可以买三只小鸡。
要求：公鸡、母鸡、小鸡都要有，一共买 100 只鸡。有哪几种买法，公鸡、母鸡、小鸡分别是多少只？

思路

此题是一个三元一次方程。
假设公鸡、母鸡、小鸡分别有 x只，y只，z只。
条件1：5x + 3y + 1/3z = 100(钱)；
条件2：x + y + z = 100(只)；得出 z = 100-x-y；
条件1*3 - 条件 2 得出：
14x + 8y = 200；y = (100-7x)/4；7x < 100 得出 x < 15；

代码

function buy(){
   for(var x=1;x < 15;x++){
       var y,z;
       y = (100-7*x)/4;
       z =  100- y - x;
       if((100-7*x)%4 == 0){
           console.log(x,y,z);
       }
       //4 18 78
       //8 11 81
       //12 4 84
   }
}
buy();

答案

百钱买百鸡一共有 3 种买法。
第一种：其中公鸡 4 只，花了 4 * 5=20 钱；母鸡 18 只，花了 18 * 3=54 钱；小鸡 78 只，花了 78/3=26 钱。鸡总数 4+18+78=100；钱总数 20+54+26=100；
第二种：其中公鸡 8 只，花了 8 * 5=40 钱；母鸡 11 只，花了 11 * 3=33 钱；小鸡 81 只，花了 81/3=27 钱。鸡总数 8+11+81=100；钱总数 40+33+27=100；
第三种：其中公鸡 12 只，花了 12 * 5=60 钱；母鸡 4 只，花了 4 * 3=12 钱；小鸡 84 只，花了84/3=28 钱。鸡总数 12+4+84=100；钱总数 60+12+28=100；
