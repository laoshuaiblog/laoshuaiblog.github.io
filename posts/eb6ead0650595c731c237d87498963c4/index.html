XShell7出现 Error Report Program has stopped working

问题：长时间没打开xshell 了，一打开就出现如下图的错误，怎么输入点击都不行，最后发现是 区域编码 问题


解决办法：

1. 打开 控制面板

2. 点击更改日期、时间或数据格式

3. 切换管理选项卡，点击更改系统区域设置

4. 取消勾选Beta

5. 重启电脑，即可正常打开xshell7
