nrm切换源和私有库下载方法

1.安装nrm（npm 源管理器）

可以在 npm源间快速切换

npm install -g nrm

2.添加公司私有库

nrm add registry http://npm.siyouku.co/

3.查看可选源

nrm ls

4.登录私有库地址输入账号密码

npm login

5.下载私有库里的包

npm i vite-plugin-aliyun-oss -D
