Java--百钱百鸡（经典算法）

百钱买百鸡的问题算是一套非常经典的不定方程的问题，题目很简单：公鸡5文钱一只，母鸡3文钱一只，小鸡3只一文钱，问公鸡，母鸡，小鸡要买多少只刚好凑足100文钱。

由题目可知这两条公式：

公鸡+母鸡+小鸡 = 100

5*公鸡+3*母鸡+小鸡/3 = 100

具体实现代码如下:

 public static void main(String[] args) {
        int gj = 0;//0-20
        int mj = 0;//0-33
        int xj = 0;
        for ( gj = 0; gj<= 20; gj++) {
            for ( mj = 0; mj<= 33; mj++) {
                xj = 100 - gj - mj;
                if (gj * 5 + mj * 3 + xj / 3 == 100 && xj % 3 == 0) {
                    System.out.println("公鸡:" + gj + "母鸡:" + mj + "小鸡:" + xj);
                }

            }
        }
