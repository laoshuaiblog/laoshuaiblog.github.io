前端JS处理后台传递的Boolean类型的获取方式

遇到的问题:

页面需要获取后端传递的boolean类型的数据, 如果不进行处理,显示true/false

function showPageList(datas) {
    $("#pageList div div").addClass("blockquote-reverse");
    $("#pageList div div").attr("style", "border: 0px none;");
    $("#bodyList").empty();
    var tableHtml = "";//拼接展示列表
    for (var key in datas) {
        var data = datas[key];
        tableHtml += '<tr>'
            + '<td>' + data.Name + '</td>'
            + '<td>' + data.EmpID + '</td>'
            + '<td>' + data.jobName + '</td>'
            + '<td>' + data.EnterpriseName + '</td>'
            + '<td id="'+data.FirstPut+'">' + (data.FirstPut == true?"是":"否") + '</td>'
            + '<td><button type="but ton" class="btn btn-warning btn-modify-standard"   onclick="update(this)">修改</button></td>'
            + '</tr>';
    }
    $("#bodyList").html(tableHtml);
}

可以看到  (data.FirstPut == true?"是":"否")  使用三元表达式 来对boolean类型的值进行处理
