LeetCode 816. 模糊坐标 做题心路历程，代码与收获

我们有一些二维坐标，如 “(1, 3)” 或 “(2, 0.5)”，然后我们移除所有逗号，小数点和空格，得到一个字符串S。返回所有可能的原始字符串到一个列表中。

原始的坐标表示法不会存在多余的零，所以不会出现类似于"00", “0.0”, “0.00”, “1.0”, “001”, "00.01"或一些其他更小的数来表示坐标。此外，一个小数点前至少存在一个数，所以也不会出现“.1”形式的数字。

最后返回的列表可以是任意顺序的。而且注意返回的两个数字中间（逗号之后）都有一个空格。

示例 1:
输入: “(123)”
输出: [“(1, 23)”, “(12, 3)”, “(1.2, 3)”, “(1, 2.3)”]
示例 2:
输入: “(00011)”
输出: [“(0.001, 1)”, “(0, 0.011)”]
解释:
0.0, 00, 0001 或 00.01 是不被允许的。
示例 3:
输入: “(0123)”
输出: [“(0, 123)”, “(0, 12.3)”, “(0, 1.23)”, “(0.1, 23)”, “(0.1, 2.3)”, “(0.12, 3)”]
示例 4:
输入: “(100)”
输出: [(10, 0)]
解释:
1.0 是不被允许的。

提示:

4 <= S.length <= 12.
S[0] = “(”, S[S.length - 1] = “)”, 且字符串 S 中的其他元素都是数字。

class Solution {
public:
    vector<string> DKER(string s)
    {
        vector<string> temp;
        temp.clear();
        for(int i = 1;i <= s.size()-1;i++)//'.'添加的位置
        {
            string substring1 = s.substr(0,i);//小数点左边
            string substring2 = s.substr(i,s.size()-i);//小数点右边
            if(substring1.size() > 1 && substring1[0] == '0')
                continue;
            if(substring2[substring2.size()-1] == '0')
                continue;
            temp.push_back(substring1 + '.' + substring2);
        }
        if(s.size() == 1 || (s.size() != 1 && s[0] != '0'))
            temp.push_back(s);
        return temp;
    }
    vector<string> ambiguousCoordinates(string s) {
        string numbers = s.substr(1,s.size()-2);
        // cout << numbers;
        vector<string> result;
        result.clear();
        for(int i = 1;i <= numbers.size()-1;i++)//','添加的位置
        {
            string substring1 = numbers.substr(0,i);
            // cout << substring1 << endl;
            vector<string> DKER1 = DKER(substring1);
            string substring2 = numbers.substr(i,numbers.size()-i);
            // cout << substring1 << endl;
            vector<string> DKER2 = DKER(substring2);
            for(int j = 0;j < DKER1.size();j++)
            {
                for(int k = 0;k < DKER2.size();k++)
                {
                    string temp = '(' + DKER1[j] + ", " + DKER2[k] + ')';
                    result.push_back(temp);
                }
            }
        }
        return result;

    }
};

1.本题先按照逗号将原数据字符串切分，再针对每个被切分的字符串使用’.'切分，两者切分方式一样，判断可行性即可，重点在如何舍去异常解，小数点前舍弃与小数点后舍弃并不一样。
