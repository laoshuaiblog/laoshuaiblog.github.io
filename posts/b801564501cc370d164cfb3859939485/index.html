[ssh] remote host key has changed

vscode 使用 ssh 连接容器时，出现这个问题。不慌，因为之前通过 8000 端口连接过该容器，后来容器被删除重新创建，导致这个问题。

不慌，先用命令行连接下

ssh asdfg@202.xx.xx.166 -p 8000

当然是连接不上

@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
@    WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!     @
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
IT IS POSSIBLE THAT SOMEONE IS DOING SOMETHING NASTY!
Someone could be eavesdropping on you right now (man-in-the-middle attack)!
It is also possible that a host key has just been changed.
The fingerprint for the ECDSA key sent by the remote host is
SHA256:2sDvjwqDXkKFRWaZXYg+zsj0fjNU4/RYwfD9CyhnC+U.
Please contact your system administrator.
Add correct host key in /home/hanjiale/.ssh/known_hosts to get rid of this message.
Offending ECDSA key in /home/hanjiale/.ssh/known_hosts:11
  remove with:
  ssh-keygen -f "/home/hanjiale/.ssh/known_hosts" -R "[202.xx.xx.166]:8000"
ECDSA host key for [202.xx.xx.166]:8000 has changed and you have requested strict checking.
Host key verification failed.

按照提示执行

ssh-keygen -f "/home/hanjiale/.ssh/known_hosts" -R "[202.xx.xx.166]:8000"

然后重新连接即可！
