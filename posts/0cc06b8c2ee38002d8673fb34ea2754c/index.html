nginx服务器启动后关不了

使用./nginx -s stop命令关闭nginx服务器，发现nginx服务没有成功停掉，进入http://127.0.0.1/还是正常显示

再次./nginx -s stop或./nginx -s quit命令，报错
nginx: [error] CreateFile() “D:\newFold\nginx-1.25.2/logs/nginx.pid” failed (2: The system cannot find the file specified)

原因

nginx启动了多个，但只关闭了一个

解决方法

命令行中输入以下内容

taskkill /f /t /im nginx.exe

强制终止运行中的 nginx.exe 进程及其所有子进程
成功停止服务器
