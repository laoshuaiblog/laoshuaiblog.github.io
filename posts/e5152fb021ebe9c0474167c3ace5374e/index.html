Free Mybatis plugin

插件名： Free Mybatis plugin
功能：

  • 提供Mapper接口与配置文件中对应SQL的导航
  • 编辑XML文件时自动补全
  • 根据Mapper接口, 使用快捷键生成xml文件及SQL标签
  • ResultMap中的property支持自动补全，支持级联(属性A.属性B.属性C)
  • 快捷键生成@Param注解
  • XML中编辑SQL时, 括号自动补全
  • XML中编辑SQL时, 支持参数自动补全(基于@Param注解识别参数)
  • 自动检查Mapper XML文件中ID冲突
  • 自动检查Mapper XML文件中错误的属性值
  • 支持Find Usage
  • 支持重构从命名
  • 支持别名
  • 自动生成ResultMap属性

CamelCase (下划线转驼峰)

快捷键：ctrl+shift+u，可以进行大小写的转换，并不能满足开发的需求。
IntelliJ IDEA提供了CamelCase很方便的工具插件，使用快捷键：alt+shift+u。
按住alt+shift再不停的按U，会把选中内容的单词的下划线转驼峰转大写等，不停的转换，会得到你想要的。

Alibaba Java Coding Guidelines

阿里巴巴代码规范检查插件。
插件名： Alibaba Java Coding Guidelines
功能： 代码规范检查。
使用方法： 在你需要检查的代上面，点击右键，选择编码规约扫描。

将会出现如下所示的检查结果，并会给出编码规范和提示：

AiXcoder Code Completer(代码提示补全)

插件名： AiXcoder Code Completer
功能： 代码提示补全插件。使用 AI 去自动提示和补全代码，比 IDEA 自带的代码补全更加智能化。

GenerateAllSetter

代码生成插件。一键生成指定对象的所有 set 方法调用代码，自动赋值，或者生成某方法的返回值，这在单元测试造假数据时非常有用。

GenerateSerialVersionUID

代码生成插件。一键为实现 Serializable 接口的类生成 SerialVersionUID。

GitToolBox（每一行代码Git的用户和日期）

Git 增强插件。在自带的 Git 功能之上，新增了查看 Git 状态、自动拉取代码、提交通知等功能。最好用的是可以查看到每一行代码的最近一次提交信息。

Key promoter X(帮助记忆快捷键)

快捷键提示插件。
插件名： Key promoter X
**功能：**当您在IDE内的按钮上使用鼠标时，键启动器X会显示您本该使用的键盘快捷键。
**使用方法：**当你点击鼠标一个功能的时候，可以提示你这个功能快捷键是什么。
比如我打断点的时候，它会提示打断点的快捷键是 Ctrl + F8：

Translation(翻译工具)

翻译的插件。
插件名： Translation
功能： 在平时的开发中，有时候对于变量的命名是否很头疼，这款插件可以帮你忙。
使用方法： 选中你要翻译的汉语或英文，点击鼠标右键，选择Translate，就会实现翻译，不用再去切换屏幕使用翻译软件翻译了。

CodeGlance(缩略图插件)

代码编辑区缩略图插件。
插件名： CodeGlance
功能： 可以快速定位代码，使用起来比拖动滚动条方便多了。

Rainbow Brackets(区分括号嵌套层级)

可以实现配对括号相同颜色，并且实现选中区域代码高亮的功能。

Nyan progress bar

基于Idea使用的漂亮进度条。把你的加载进度条全都变成彩虹和猫咪

SequenceDiagram（一键生成时序图）

插件名： SequenceDiagram
功能： 生成简单的方法时序图，方便我们阅读代码，特别是在代码的调用层级比较多的时候。
使用方法： 选中方法名（注意不要选类名），然后点击鼠标右键，选择 Sequence Diagram 选项即可！

SonarLint（帮你优化代码）

插件名： SonarLint
功能： SonarLint 帮助你发现代码的错误和漏洞，就像是代码拼写检查器一样，SonarLint 可以实时显示出代码的问题，并提供清晰的修复指导，以便你提交代码之前就可以解决它们。

炫酷Sao操作

插件名： Power mode II
