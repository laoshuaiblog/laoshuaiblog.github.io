vim | vim的快捷命令行

快捷进入shell界面 -> :nnoremap <F8> :sh<CR> -> 绑定到了F8

:nnoremap <F8> :sh<CR>

快捷执行 -> :nnoremap <F5> :wa<CR>:!g++ % -o a.out && ./a.out<CR> -> 绑定到了F5

:nnoremap <F5> :wa<CR>:!g++ % -o a.out && ./a.out<CR>

快捷保存 -> :noremap <F4> :wa<CR> -> 绑定到了F4

 :nnoremap <F4> :wa<CR>
