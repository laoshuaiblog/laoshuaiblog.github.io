2.2、all

  • 给定的可迭代参数 中的元素，所有元素是否都为 TRUE，返回 True，否则（只要有一个为假）返回 False。
  • 元素除了是 0、空、None、False 外都算 True。
  • 空元组、空列表返回值为True

等价于：

#等价于
def all(iterable):
    for element in iterable:
        if not element:
            return False
    return True

示例：

print(all([1,2]))
print(all([1,2,0]))

输出：
True
False

#空元组、空列表返回值为True
print(all(("1", ""))) #含有空元素，False
print(all(("")))      #True
print(all(()))        #True

print(all(["1", ""])) #含有空元素，False
print(all([""]))      #False
print(all([]))        #True,空列表True

2.3、any

  • 用于判断给定的可迭代参数 中的元素，有一个元素为 TRUE，则返回 True，否则(即全为Falase)返回 False。
  • 元素除了是 0、空、FALSE 外都算 TRUE

等价于：

#any函数等价于
def any(iterable):
    for element in iterable:
        if element:
            return True
    return False

示例：

print(any(["a", "b", "c"]))      # True
print(any(["a", "b", ""]))       # True，存在一个元素为True,返True
print(any([0, '', False]))       # False，全部为空
print(any(""))                   # False, FALSE，0, "" 三元素不能在元祖中共存
print(any(("a", "b", "c")))      # True
print(any(("a", "b", "")))       # True，存在一个元素为True,返True

print(any(""))      # False
print(any([""]))    # False
print(any(()))      # False
print(any([]))      # False

2.59、sorted

对所有可迭代的对象进行排序操作

print(sorted([5,3,2,4,1]))              #默认为升序
print(sorted([5,3,2,4,1],reverse=True)) #降序

输出：
[1, 2, 3, 4, 5]
[5, 4, 3, 2, 1]

#默认以key排序
dicAge={"e":1,"a":5,"c":3,"b":2,"d":4,}
print(sorted(dicAge))

输出：
[‘a’, ‘b’, ‘c’, ‘d’, ‘e’]

#根据values排序，输出key
dicAge={"e":1,"a":5,"c":3,"b":2,"d":4,}
print(sorted(dicAge,key=lambda key:dicAge[key]))

输出：
[‘e’, ‘b’, ‘c’, ‘d’, ‘a’]

dicAge={"e":1,"a":5,"c":3,"b":2,"d":4,}
print(sorted(zip(dicAge.keys(),dicAge.values())))

2.67、 zip

用于将可迭代的对象作为参数，将对象中对应的元素打包成一个个元组，然后返回由这些元组组成的对象，这样做的好处是节约了不少的内存。
可以使用 list() 转换来输出列表。
如果各个迭代器的元素个数不一致，则返回列表长度与最短的对象相同，利用 * 号操作符，可以将元组解压为列表。

print(zip(("a","b","c"),(1,2,3)))
print(list(zip(("a","b","c"),(1,2,3))))

输出：
<zip object at 0x000000FE89EA9248>
[(‘a’, 1), (‘b’, 2), (‘c’, 3)]

p={"name":"chuhe","age":"18"}
print(list(zip(p.keys(),p.values())))

输出：
[(‘name’, ‘chuhe’), (‘age’, ‘18’)]

print(list(zip("hello","12345")))

输出：
[(‘h’, ‘1’), (‘e’, ‘2’), (‘l’, ‘3’), (‘l’, ‘4’), (‘o’, ‘5’)]
