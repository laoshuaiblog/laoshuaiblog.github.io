文章目录

问题

public class People {
    int m = 10,n; // 1
    n = 200; // 2
    void f(){
        if(m == n)
            n  += m; // 3
        else
            n = n - m; // 4
    }

}

这道题问的是，下列哪行代码有错误，一开始并没有发现问题的所在，放了编辑器里面，发现报错unexpected token，查阅资料后发现

class下只能定义方法，以及变量等。但不能直接编写逻辑代码，应该把这些代码定义在一个方法里面

所以解决办法很简单，只要把对n的赋值方法方法f()中就可以解决报错啦，如下

public class People {
    int m = 10,n; // 1
   
    void f(){
    	 n = 200; // 2
        if(m == n)
            n  += m; // 3
        else
            n = n - m; // 4
    }

}
