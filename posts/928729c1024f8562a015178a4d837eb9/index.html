求出所有的水仙花数。所谓水仙花数是指一个三位数，其各位数字的立方和等于该数字本身。比如153是一个水仙花数，因为153=1^3+5^3+3^3。

#include <stdio.h>
int main(void) 
{
    int a,b,c;
    for (a=1;a<10;a++)
    for (b=0;b<10;b++)
    for (c=0;c<10;c++)
    if (a*100+b*10+c==a*a*a+b*b*b+c*c*c)
    printf("%d",a*100+b*10+c);
    return 0;
}
