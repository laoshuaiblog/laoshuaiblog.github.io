小球反弹-C++

一个

#include <iostream>

using namespace std;

int main()
{
    int n,sum=0;
    cin>>n;
    sum=n;
    for(int i=2;i<=10;i++)
    {
        sum+=n;
        n=n/2;
    }
    cout<<sum;
    return 0;
}


小球从n米高度落下，每次落地后反弹为原来高度的一半，再落下，球在10次落地后，共经过了多少米？
