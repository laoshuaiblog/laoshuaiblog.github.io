C语言——水仙花数

今日笔者突然有了兴致，便写一个很简单的适合于C语言初学者的程序。

水仙花数定义：一个三位数i它的百位十位个位分别为a,b,c。若是i=a^3+b^3+c^3那么该数称为水仙花数。

输出100-999以内的水仙花数

代码如下：

#include<stdio.h>
int main()
{
    for(int i=100;i<1000;i++)    ——循环条件
    {
        int a,b,c;
        a=i/100;
        b=(i-100*a)/10;
        c=i-100*a-10*b;
        if(i==a*a*a+b*b*b+c*c*c)  ——判断语句
        printf("%d\t",i);
    } 
    return 0;
}

执行可以得到结果为 153 370 371 407

很简单的程序适合初学者掌握判断循环语句的书写。
