水仙花数-C语言

所谓水仙花数，是指一个三位数，其各位数字的立方和等于该数本身。例如：153是一个水仙花数，因为153=1*1*1+5*5*5+3*3*3。请求出给定范围n内的水仙花数
#include <stdio.h>
#include <stdlib.h>

int main()
{
    int n,a,b,c,i;
    scanf("%d",&n);
    for(i=153;i<=n;i++)
    {
        a=i/100;
        b=i%100/10;
        c=i%10;

        if(i==a*a*a+b*b*b+c*c*c)
           printf("%d ",i);

    }
    return 0;
}
