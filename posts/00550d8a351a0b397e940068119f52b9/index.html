Windows下后台运行、关闭jar的命令

1.启动 创建bat文件 输入内容如下（java.jar需要改成自己的）

javaw 代表后台运行，不会出现黑框框

@echo off

start javaw -Dfile.encoding=UTF-8 -jar java.jar

exit

运行 jar包，并输出日志

@echo off
%1 mshta vbscript:CreateObject("WScript.Shell").Run("%~s0 ::",0,FALSE)(window.close)&&exit
java -jar java.jar > D:\test\log\server.log 2>&1 &
exit

2.关闭 创建bat文件

杀死java进程

@echo off

taskkill /f /im javaw.exe

exit
