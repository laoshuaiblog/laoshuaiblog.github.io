前端去重方法实例

const array = [
{
    type: 11111,
},
{
    type: 11111,
},
{
    type: 11111,
},
];

如上数据进行去重，用以下方法
 

form = [...new Set(array.map(item => item.type))].map(id => ({ type: id }));

form得到的数据就是去重之后的数据
