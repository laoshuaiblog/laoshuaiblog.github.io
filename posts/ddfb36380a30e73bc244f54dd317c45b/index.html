ajax返回boolean的坑

前台ajax请求，后台返回boolean（代码如下）

$.ajax({
	type : "post",
	data : {
			"xxx" : xxx
	},
	url: "/xxx/xxx.do",
	success: function (result) {
		console.log(result);
		console.log(typeof result);
		if (result) {
			alert(true);
		} else {
			alert(false);
		}
	}
});
@RequestMapping("test")
@ResponseBody
public boolean test() {
	return false;
}

后台返回的Boolean类型，ajax是不认的，会简单的认为是string类型。如果返回 false 也会执行 if (result) 中的逻辑而不是走 else，因为 if('false') 是等于 true 的。

需要将代码修改为如下

$.ajax({
	type : "post",
	data : {
			"xxx" : xxx
	},
	url: "/xxx/xxx.do",
	success: function (result) {
		console.log(result);
		console.log(typeof result);
		if (result == "true") {
			alert(true);
		} else {
			alert(false);
		}
	}
});
