python数据分析之pandas软件包

 创建新文件

  1. 引用pandas软件包 
    #pd是pandas简写
import pandas as pd
  2. 创建excel文件 
    df.to_excel('文件路径')
  3. 列名索引 
    df.set_index('列名')
  4. 写入数据（默认为空） 
    pd.DataFrame('列名1':['数据1','数据2','数据3'],'列名2':['数据1','数据2','数据3'])
  5. 写入excel,运行 
    print('Done!')

    读取数据 

  1.  读取excel的行数和列数 
    print(df.shape)
  2. 读取列名 
    print(df.columns)
  3. 读取前x行数据 
    print(df.head(x))
  4. 读取后x行数据 
    print(df.tail(x))
  5. 设置头部信息 
     df.columns['列名1','列名2','列名3']
