OPLS-DA的S-plot图可视化

OPLS-DA的S-plot图，横坐标表示主成分与代谢物的协方差，纵坐标表示主成分与代谢物的相关系数，越靠近右上角和左下角的代谢物表示其差异越显著，红色的点表明这些代谢物的VIP值大于等于1，蓝色的点表示这些代谢物的VIP值小于1。

library(testpackage)
s_plot_data <- t(s_plot_data)
data <- data.frame('Secondary_ID' = c('A1','A2','A3','A4',
                                      'A5','A6','A7','A8',
                                      'B1','B2','B3','B4',
                                      'B5','B6','B7','B8'),
                   'Class_ID' = rep(c('A','B'),each = 8))
data <- data.frame(data,s_plot_data)
s_plot(data)
