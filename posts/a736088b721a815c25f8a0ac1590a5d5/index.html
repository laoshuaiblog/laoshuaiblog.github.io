node.js常用的命令

node.js常用的命令包括：
安装包。使用`npm install <包名>@版本号`安装指定版本的包，或者使用`npm install <包名>`安装包到当前目录。
卸载包。使用`npm uninstall <包名>`卸载包。
全局安装包。使用`npm install <包名> -g`全局安装包。
查看包信息。使用`npm info <包名>`查看远程包信息，或者使用`npm ls <包名>`查看本地包信息。
设置npm配置。使用`npm config set <配置名> <配置值>`设置npm的配置。
切换npm镜像源。使用`npm config get registry`查看当前的npm镜像源，然后使用`npm config set registry=https://registry.npm.taobao.org`将镜像源切换为淘宝。
以上命令可以帮助开发者更好地管理和使用node.js项目中的包。
