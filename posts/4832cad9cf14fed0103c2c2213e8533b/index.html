Docker部署nginx

搜索镜像

docker search nginx

下载拉取nginx镜像

docker pull nginx

查看镜像

docker images

启动容器

docker run -d --name nginx01 -p 3344:80 nginx

外部端口需要在服务器安全组中设置，使用docker镜像nginx以后台模式启动一个容器，并将容器命名为nginx01

-d 是后台访问

-p 指定端口映射，格式为：主机(宿主)端口:容器端口

--name 为容器指定一个名称

查看容器

docker ps

测试访问

curl 127.0.0.1:3344

浏览器输入服务器地址加端口号出现下图即安装成功

进入容器修改界面

docker exec -it nginxd /bin/bash

查看nginx安装位置

whereis nginx

退出容器修改界面

exit
