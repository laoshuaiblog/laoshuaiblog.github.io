3.1.2.修改web.xml，添加内容

<?xml version="1.0" encoding="UTF-8"?>

         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"

    <servlet>
        <servlet-name>springmvc</servlet-name>
        <servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>
        <!--关联一个springmvc的配置文件:【servlet-name】-servlet.xml-->
        <init-param>
            <param-name>contextConfigLocation</param-name>
            <param-value>classpath:springmvc-servlet.xml</param-value>  <!--要写一个配置文件-->
        </init-param>
        <!--启动级别-1-->
        <load-on-startup>1</load-on-startup>
<!--   the above should be ignored at this time     -->

    </servlet>

    <!--/ 匹配所有的请求；（不包括.jsp）-->
    <!--/* 匹配所有的请求；（包括.jsp）-->
    <servlet-mapping>
        <servlet-name>springmvc</servlet-name>  <!--所有请求将会经过-->
        <url-pattern>/</url-pattern>
    </servlet-mapping>

</web-app>

3.1.4.添加处理映射器、处理适配器、视图解析器

完整代码位于3.1.7

<!--配置三个类：可以直接复用-->

    <!--  处理映射器  -->
    <bean class="org.springframework.web.servlet.handler.BeanNameUrlHandlerMapping"/>

    <!--  处理器适配器 -->
    <bean class="org.springframework.web.servlet.mvc.SimpleControllerHandlerAdapter"/>

    <!--视图解析器:DispatcherServlet给他的ModelAndView-->
    <bean class="org.springframework.web.servlet.view.InternalResourceViewResolver" id="InternalResourceViewResolver">
        <!--前缀-->
        <property name="prefix" value="/WEB-INF/jsp/"/>
        <!--后缀-->
        <property name="suffix" value=".jsp"/>
    </bean>

3.1.5.写具体业务操作

编写我们要操作业务Controller ，要么实现Controller接口，要么增加注解；需要返回一个ModelAndView，装数据，封视图；

package com.kuang.controller;

import org.springframework.web.servlet.ModelAndView;
import org.springframework.web.servlet.mvc.Controller;

import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;

public class HelloController implements Controller {    //导入的Controller选择接口类型
//    实现了一个controller接口，就能处理request和response
    public ModelAndView handleRequest(HttpServletRequest request, HttpServletResponse response) throws Exception {
        //ModelAndView 模型和视图
        ModelAndView mv = new ModelAndView();

        //封装对象，放在ModelAndView中。Model
        mv.addObject("msg","HelloSpringMVC!");
        //封装要跳转的视图，放在ModelAndView中
        mv.setViewName("hello"); //: /WEB-INF/jsp/hello.jsp
        return mv;
    }
}

3.1.6.要跳转的jsp页面，显示ModelandView存放的数据，以及正常页面

新建名为jsp的文件夹，hello.jsp文件

3.1.7.将自己写的类交给SpringIOC容器，注册bean

加粗部分是本部分新增的，其余为之前的两部分

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans
       http://www.springframework.org/schema/beans/spring-beans.xsd">

<!--配置三个类：可以直接复用-->

    <!--  处理映射器  -->
    <bean class="org.springframework.web.servlet.handler.BeanNameUrlHandlerMapping"/>

    <!--  处理器适配器 -->
    <bean class="org.springframework.web.servlet.mvc.SimpleControllerHandlerAdapter"/>

    <!--视图解析器:DispatcherServlet给他的ModelAndView-->
    <bean class="org.springframework.web.servlet.view.InternalResourceViewResolver" id="InternalResourceViewResolver">
        <!--前缀-->
        <property name="prefix" value="/WEB-INF/jsp/"/>
        <!--后缀-->
        <property name="suffix" value=".jsp"/>
    </bean>

    <!--Handler-->
    <bean id="/hello" class="com.kuang.controller.HelloController"/>
    <!--请求hello时是从spring而不是从web.xml调用-->
</beans>

3.1.8.配置tomcat，启动测试

如果报错404，需要在File - Project Construction - Artifacts - ‘springmvc-02-hello:war exploded’ - WEB-INF目录下新建directory：命名为lib - 添加Library Files全选其中有的所有内容，最后点击apply

http://localhost:8080/springmvc_02_hello_war_exploded/hello

3.1.9.可能遇到的问题：访问出现404，排查步骤

  1. 查看控制台输出，看一下是不是缺少了什么jar包。
  2. 如果jar包存在，显示无法输出，就在IDEA的项目发布中，添加lib依赖！
  3. 重启Tomcat 即可解决！
